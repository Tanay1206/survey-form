# Tanay's Friendship Feedback Form

A responsive and interactive personal survey application designed to capture feedback and route it to a live database. This project serves as the frontend interface, bridging a clean user experience with backend data storage.

##  Features

* **Responsive Design:** Built with HTML and CSS to ensure the form is accessible and looks great on desktop and mobile devices.
* **Interactive Validation:** Utilizes vanilla JavaScript to handle form validation (such as age constraints) before submission.
* **Live API Integration:** Form data is submitted via a POST request directly to a live backend service hosted on Render, which securely stores the responses in MongoDB.

##  Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend Integration:** Node.js/Express and MongoDB (Hosted on Render)

##  Getting Started

Because this project uses native web technologies, there are no complicated installation steps or package managers required.

### How to Run Locally

1. Clone the repository to your local machine:
   ```bash
   git clone [https://github.com/Tanay1206/survey-form.git](https://github.com/Tanay1206/survey-form.git)

2. Navigate into the project folder.

3. Simply double-click the index.html file to open it in any web browser, or open it using an extension like VS Code's "Live Server".

Note: The form is fully functional locally and will successfully send form submissions to the live backend database upon clicking submit.
